# Social Content Knowledge Base

Lessons learned from real content analysis. Reference when creating or reviewing social media content.

## Purpose

- Accumulate patterns from draft vs production comparisons
- Track anti-patterns to avoid
- Platform-specific insights from actual content performance

## Files

| File | Platform |
|------|----------|
| `facebook-lessons.md` | Facebook page/group posts |
| `linkedin-lessons.md` | LinkedIn professional content |
| `twitter-lessons.md` | Twitter/X posts and threads |
| `instagram-lessons.md` | Instagram feed, stories, reels |
| `general-patterns.md` | Cross-platform patterns |

## Adding New Lessons

### When to Add
- After analyzing draft vs production content
- After reviewing content performance metrics
- When discovering new platform-specific insights

### Pattern Format
```markdown
## ✅ Pattern: [Short Descriptive Name]
- **Source**: [YYYY-MM-DD] | [Content Type/Topic]
- **Context**: [Brief situation description]
- **Lesson**: [What to do]
- **Apply when**: [Trigger conditions]
```

### Anti-Pattern Format
```markdown
## ❌ Anti-Pattern: [Short Descriptive Name]
- **Source**: [YYYY-MM-DD] | [Content Type/Topic]
- **Issue**: [What went wrong]
- **Fix**: [How to correct]
- **Why matters**: [Impact explanation]
```
