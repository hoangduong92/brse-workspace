## Quick Reference

### Git Commands
```bash
git add .
git commit -m "message"
git push origin main
```

### Next.js New Project
```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

### Deploy to Vercel
```bash
# Just push to GitHub, Vercel auto-deploys
# Or: npx vercel
```

### Supabase CLI
```bash
npx supabase init
npx supabase start
```
