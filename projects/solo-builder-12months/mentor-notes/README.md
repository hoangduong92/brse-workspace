# Mentor Notes

Session summaries và insights từ AI Mentor trong suốt 12-month journey.

## Structure

```
mentor-notes/
├── README.md
├── phase-1/
│   ├── week-01-02.md
│   ├── week-03-04.md
│   └── ...
├── phase-2/
└── ...
```

## How to Use

1. **Sau mỗi session học**: Mentor ghi summary vào file tuần tương ứng
2. **Cuối mỗi phase**: Mentor viết phase retrospective
3. **Khi cần review**: Đọc lại notes để refresh kiến thức

## Note Format

```markdown
## Session: YYYY-MM-DD

### Topics Covered
- Topic 1: [brief description]

### Key Insights
- Insight 1

### Questions Asked (Socratic)
- Q1: [question] → [how learner answered]

### Knowledge Gaps Identified
- Gap 1: [description]

### Next Session Focus
- Focus 1
```
