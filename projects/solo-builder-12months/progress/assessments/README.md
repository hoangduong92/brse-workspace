# Phase Assessment System

> Ship fast + Learn deep = Real engineer

## Rules

| Setting | Value |
|---------|-------|
| Pass threshold | 70% |
| Format | Quiz + Explain-to-me |
| Timing | End of each phase |
| Retake | Unlimited (learn from mistakes) |

## Assessment Types

### 1. Quiz (Quick Knowledge Check)
- 10-15 questions per phase
- Mix of concepts + "what happens if..."
- Focus on WHY, not syntax

### 2. Explain-to-me (Deep Understanding)
- Pick random code from your shipped products
- Explain line-by-line to mentor
- Must answer follow-up questions
